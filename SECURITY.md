# Security Policy

Deutsch: [SECURITY.DE.md](SECURITY.DE.md)

## Supported Versions

Only the current release line receives security updates.

## Reporting a Vulnerability

Please use GitHub Security Advisories for private vulnerability disclosure:

- https://github.com/mildman1848/SmartSkip/security/advisories/new

Include the affected Kodi version, addon version, reproduction steps, and expected impact.
