# SmartSkip

English: [README.md](README.md)

SmartSkip ist ein Kodi-Dienst-Addon für Kodi Omega und neuer, das Intros, Recaps, Outros, Vorschauen und Credits auf Basis von EDL-Dateien automatisch oder manuell überspringen kann.

Die Segmenterkennung nutzt das eingebettete `intro-skipper`, sodass keine externen Python-Pakete benötigt werden.

## Funktionen

- automatisches und manuelles Überspringen während der Wiedergabe
- Unterstützung für Intros, Recaps, Outros, Vorschauen und Credits
- eingebettete `intro-skipper`-Integration
- zweisprachige Dokumentation und Lokalisierung
- skin-kompatibler Wiedergabedialog mit Fortschrittsanzeige

## Installation

1. Repository oder ZIP-Release herunterladen.
2. In Kodi `Add-ons` -> `Aus ZIP-Datei installieren` öffnen.
3. Das SmartSkip-ZIP auswählen.
